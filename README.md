Medora: Medicine Availability Tracker
Medora is a localized healthcare solution designed to bridge the gap between patients and pharmacies. It allows users to search for specific medicines, check real-time stock availability, and find the nearest pharmacies using geolocation.

🚀 Features
Smart Search: Find medicines by name with real-time availability status.

Geospatial Discovery: Locates pharmacies within a specific radius using the Haversine formula.

Admin Dashboard: Secure inventory management for pharmacy owners to update stock and pricing.

JWT Authentication: Secure login system for administrators and pharmacy staff.

Responsive UI: Optimized for both mobile and desktop users.

🛠️ Tech Stack
Frontend: React.js (Vite), Tailwind CSS, Lucide Icons.

Backend: FastAPI (Python), Motor (Async MongoDB driver).

Database: MongoDB.

Authentication: JWT (JSON Web Tokens) with Bcrypt password hashing.
